## Features

- User management and selection
- Add, edit, and remove pets
- Upload pet profile pictures
- Upload and preview medical files
- Carousel display of pets
- Responsive UI with modals for actions
- Simple file-based backend using JSON

---

## Tech Stack

**Frontend:**

- React 18
- TypeScript
- Tailwind CSS
- React Hooks & Context

**Backend:**

- Node.js & Express
- TypeScript
- Multer for file uploads
- CORS for cross-origin requests
- JSON file-based database (`pets.json` and `users.json`)
